# Harbor Database Schema Diagram

Interactive entity relationship diagram (ERD) for the [Harbor](https://goharbor.io/) container registry database.

## Live Demo

[https://harbor-db-diagram.resources.bupd.xyz](https://harbor-db-diagram.resources.bupd.xyz/)

## About

This tool visualizes Harbor's PostgreSQL database schema, showing:

- All database tables and their columns
- Primary keys and foreign key relationships
- Data types and constraints
- Entity relationships between tables

Built using [DBGate](https://dbgate.org/), an open-source database management tool.

## Harbor

[Harbor](https://github.com/goharbor/harbor) is an open-source container registry that stores, signs, and scans container images for vulnerabilities. It is a CNCF graduated project.

## Use Cases

- Understanding Harbor's data model
- Database documentation reference
- Development and debugging
- Migration planning
